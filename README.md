# MetaMirror (Metadata Project)

MetaMirror is a metadata-first local file-management toolkit with a CLI, policy layer, proposal workflow, and experiment harness.

## Project Layout

- `metamirror/`: core CLI and metadata logic
- `experiments/`: experiment runners and figure-data preparation
- `experiment_results/`: generated outputs
- `tests/`: test suite
- `demo/`: lightweight demo artifacts

## Local-Only Directories

Some personal or non-public work directories are kept local and ignored by git.
